## Features

- Cross-platform.

## Installation

### Dependences
- ***Ubuntu***  
```bash
sudo apt install qt5-default build-essential -y
```
- ***Windows***  
    - Qt Creator: use clipshare.pro
    - Visual Studio: use clipshare.sln (need vs-qt plugin)

### Build
- ***Ubuntu***  
```bash
qmake && make
```
- ***Windows***  
Build with IDE.
